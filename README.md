## Week 2 Project: Employee Records – Excel Formulas Practice

📊 A hands-on Excel project built while learning data analytics fundamentals (Week 2)

## 📌 About
This project uses a sample Employee Records dataset to practice combining
multiple Excel formulas — text manipulation, lookups, and date functions —
to clean, transform, and enrich raw HR data.

## 🎯 Objective
Practice building calculated/derived columns from raw employee data using
real-world formulas commonly used in data analytics and reporting.

## 🗂️ Dataset
- Sample Employee Records (15 employees)
- Columns: Employee ID, Name, Department, Manager, Hire Date, Address, Phone, Email

## 🛠️ Tools
- Microsoft Excel

## 📚 What I Learned
- How to chain multiple text functions to clean and standardize data
- Difference between exact-match and approximate-match VLOOKUP
- Using DATEDIF and NETWORKDAYS for date-based calculations
- Debugging formula errors (e.g. blank-cell errors when formulas are dragged too far)
